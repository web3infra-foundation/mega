# Libra Commands

This section provides documentation for all Libra commands.

## Command List

### [init](init)

Initialize a new Libra repository with optional support for separate git directory.

**Options:**
- `--bare`: Create a bare repository (default: `true`)
- `--separate-git-dir=<path>`: Store version control data in specified path
- `--workdir=<path>`: Specify working directory path (for non-bare repositories)

## Getting Help

To get help for any command, use the `--help` flag:

```bash
libra <command> --help
```
