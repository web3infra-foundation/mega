---

**URL:**

    ```bash
    GET /api/v1/blob
    ```

**Request Params:**

| **Field** | **Required** | **Type** | **Location** | **Description** |
| --------- | ------------ | -------- | ------------ | --------------- |
| path      | Y            | string   | Query        | blob file path  |



**Response Params:**
1. Common

| **Field** | **Type** | **Description**          |
| --------- | -------- | ------------------------ |
| data      | string   | Blob content with string |

**Request Example:**

```json
GET /api/v1/blob?path=/third-party/.mega_cedar.json
Content-Type: application/json
```
